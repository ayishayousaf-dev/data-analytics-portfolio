## How This Portfolio Is Organized
Each project folder contains:
- A `README.md` with business problem, dataset description, approach, and key insights
- Source data or code
- For Power BI projects: DAX measure documentation and reproduction steps (screenshots to be added once dashboards are built in Power BI Desktop)

## Note on Power BI Files
`.pbix` files are binary and not previewable directly on GitHub. Each Power BI project folder includes the underlying dataset (or dataset source, for public datasets), a dashboard screenshot, and full DAX documentation so the logic is reviewable without opening Power BI Desktop.
